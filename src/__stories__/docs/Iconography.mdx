## Sizing

| Size | Pixels | Use case |
| --- | --- | --- |
| `sm` | 14px | Inline with body text |
| `md` | 16px | Default UI control |
| `lg` | 20px | Header or prominent action |

```tsx
import { Settings } from 'lucide-react';

<Settings aria-hidden="true" height={16} width={16} />
```

## Accessibility

- Decorative icons should use `aria-hidden`.
- Standalone icon buttons need an accessible label on the button.
- Keep Lucide's default stroke width unless the design explicitly calls for a variant.
