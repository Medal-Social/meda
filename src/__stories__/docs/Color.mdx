# Color

Meda's color system has three layers. Layer 0 primitives are constant; Layer 1 semantic tokens flip between light and dark; Layer 2 shell tokens specialize the application chrome.

## When to use which

- Use semantic tokens such as `bg-card`, `text-foreground`, and `border-border` for product UI.
- Use primitive ramps only when the exact stop matters, such as brand moments or chart series.
- Avoid raw hex values in component code.

## Layer 0: Primitives

The brand ramp is the spine of the system. Every ramp uses the same stop convention where the source tokens define it.
