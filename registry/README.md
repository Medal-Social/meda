# Meda Registry

Open-source shadcn-compatible registry scaffold for Meda.

Current purpose:
- define the `@meda` namespace structure
- host installable Meda shell items
- document private-first and public-later registry usage

Current items:
- `meda-shell`
- `meda-shell-state`
- `meda-workbench-layout`

This registry now ships valid item JSON plus inline installable source stubs. It is ready for local validation and for a later hosted public endpoint.
